Web App

MoMento

MoMento is a project collaboration and meeting minutes web application built to help teams organize workspaces, document meetings, assign action items, and keep project follow-up visible from one central dashboard.

The platform is designed around real project workflows. Users can create project workspaces, invite team members, manage pending invitations, customize project identity, and switch between active projects from a clean workspace layout.

Inside each project, MoMento provides a dashboard for recent activity, meeting documentation, upcoming meetings, and open action items. The interface combines structured tables, filters, status badges, user avatars, and responsive layouts to keep collaboration clear and easy to scan.

Meeting minutes can include agenda items, discussion points, decisions, attendees, author information, location, meeting time, and linked action items. Minutes can be kept as drafts, published to the team, viewed in a document-style layout, and exported as PDF for sharing or archiving.

The action item module extends MoMento beyond simple note taking. It supports list and board views, assignee filtering, statuses, priorities, due dates, labels, story points, estimates, issue types, sprint grouping, parent-child task links, and soft-delete workflows.

Key features include:
  • Project workspace creation and management
  • Team invitations and member collaboration
  • Meeting minutes with agenda, notes, and decisions
  • Published and draft MoM workflows
  • Action item tracking with list and board views
  • Sprints, labels, priorities, estimates, and due dates
  • Role-aware project access and protected routes
  • PDF export for meeting minutes
  • Responsive light and dark UI with Angular Material components
  • Project backup and restore tooling

The product focuses on replacing scattered meeting notes and manual follow-up tracking with a structured, searchable, team-oriented workflow. It gives teams a clearer path from discussion to decision to accountable action.

Tech stack: Angular, TypeScript, Supabase, Tailwind CSS, Angular Material, RxJS, Editor.js, ngx-quill, ngx-editor, and Node.js project backup scripts.

MoMento is a strong example of a real-world SaaS-style web application focused on productivity, collaboration, structured project documentation, and polished user experience.
